## Install Git Using A Package Manager

A *package manager* is a command line tool that allows you to install and manage software. You can use a package manager to install Git and other software.

The package manager that we'll be using for this guide is called [Homebrew](https://brew.sh/). It's a command-line installer for MacOS, which means that you'll have to be running a macOS in order to use. The examples provided here are utilizing macvscodeOS Catalina version 10.15.5.


## Step 1: Install Homebrew


<img src="./images/brew-1.png" width="500"/>

Open your terminal and type <code>brew</code>.

The result should look like the following:

<img src="./images/brew-2.png" width="300"/>

## Step 2: Install Git

With Homebrew installed, you are now ready to install Git. Open a terminal window and enter <code>brew install git</code>

To verify that Git is installed on your system, type <code>git --version</code>
